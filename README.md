# create-ts-canvas

create canvas art project template using typescript

[![npm Package Version](https://img.shields.io/npm/v/create-ts-canvas)](https://www.npmjs.com/package/create-ts-canvas)

## Usage

Using `npm init`

```bash
npm init ts-canvas
```

or using `npx`

```bash
npx -y create-ts-canvas
```
